사용 방법 : 
1. dist.zip, dist.z01, dist.z02 을 다운받는다. (분할압축 되어있다.)
2. 압축을 푼다.
3. pdf2png를 실행한다.
4. 파일선택 버튼을 눌러 pdf파일을 선택한다.
5. 변환 시작 버튼을 누른다.
6. 변환이 완료된다.
