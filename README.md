이 프로그램은 개인용 및 비상업적 용도로만 사용 가능합니다.


사용 방법 : 
1. pdf2png.zip 을 다운받는다.
2. 압축을 푼다.
3. pdf2png를 실행한다.
4. 파일선택 버튼을 눌러 pdf파일을 선택한다.
5. 변환 시작 버튼을 누른다.
6. 변환이 완료된다.


60MB 기준, 100페이지 당 1~2분정도 걸린다.
